Some examples of the %SYSTEMWEB%.PhotoGalleryPlugin in action.


---++ Example 1

Explicitly select images.

<verbatim class="tml">
%PHOTOGALLERY{ "IMG_1262.jpg,IMG_1265.jpg" }%
</verbatim>

Produces:

%PHOTOGALLERY{ "IMG_1262.jpg,IMG_1265.jpg" }%


---++ Example 2

Select a range of images.

<verbatim class="tml">
%PHOTOGALLERY{ "IMG_1265.jpg..IMG_1324.jpg" }%
</verbatim>

Produces:

%PHOTOGALLERY{ "IMG_1265.jpg..IMG_1324.jpg" }%


---++ Example 3

Selecting an image (attachment) that does not exist.

<verbatim class="tml">
%PHOTOGALLERY{ "doesnotexist" }%
</verbatim>

Produces an error:

%PHOTOGALLERY{ "doesnotexist" }%


---++ Example 4

A gallery with all the images (attachments) not already used in the the galleries above.

<verbatim class="tml">
%PHOTOGALLERY{ remaining="on" }%
</verbatim>

Produces:

%PHOTOGALLERY{ remaining="on" }%


---++ Example 5

Using images (attachments) from another topic.

<verbatim class="tml">
%PHOTOGALLERY{ topic="System.PhotoGalleryPlugin" }%
</verbatim>

Produces:

%PHOTOGALLERY{ topic="System.PhotoGalleryPlugin" }%


---++ Example 6

Showing a single, random image without decorations, floating to the right of a text.

<verbatim class="tml">
%PHOTOGALLERY{ topic="System.PhotoGalleryPlugin" random="1" width="1" float="right" size="100" thumbcap="" }%
blablabla blabla...
<div style="clear: both;"></div>
</verbatim>

Produces:

%PHOTOGALLERY{ topic="System.PhotoGalleryPlugin" random="1" width="1" float="right" size="100" thumbcap="" }%
blablabla blabla blablabla blabla blablabla blabla blablabla blabla blablabla
blabla blablabla blabla blablablablabla blablabla blabla blablabla blabla
blablabla blabla blablabla blabla blablabla blabla blablabla blabla blablabla
blabla blablabla blabla blablabla blabla
<div style="clear: both;"></div>


---++ Example 7

Select all images with a name that start with "notjepg".

<verbatim class="tml">
%PHOTOGALLERY{ "/^notjpeg/" }%
</verbatim>

Produces:

%PHOTOGALLERY{ "/^notjpeg/" }%


---++ Example 7

<verbatim class="tml">
%PHOTOGALLERY{ "IMG_1262.jpg,notjpeg_withmeta.png" lazy="off" }%
</verbatim>

Produces:

%PHOTOGALLERY{ "IMG_1262.jpg,notjpeg_withmeta.png" lazy="off" }%


---++ Example 8

%TABPANE%
%TAB{"Hello"}%
We can have a gallery initially hidden, for example in a tab. Switch to the "Gallery" tab to view it.
%ENDTAB%
%TAB{"Gallery"}%
%PHOTOGALLERY%
%ENDTAB%
%ENDTABPANE%


---++ List of all Galleries

We can generate a list of all topics that use the =%<nop>PHOTOGALLERY%= macro in the [[Sandbox.WebHome][Sandbox]] web:

<verbatim class="tml">
%SEARCH{ "%PHOTOGALLERY" web="Sandbox" type="literal" nonoise="on" format="   * [[$item][$item]]" }%
</verbatim>

Produces:

%SEARCH{ "%PHOTOGALLERY" web="Sandbox" type="literal" nonoise="on" format="   * [[$item][$item]]" }%
